# Ganpati Agent 🐘

Ye skill aapke project ki saari files ka index banata hai aur intelligent guidance provide karta hai. Har command se pehle index read karta hai aur related files ko efficiently handle karta hai.

## Main Features:

### 1. Project File Indexing
- Automatically creates and maintains project file index
- Understands file relationships and dependencies
- Quick file navigation and search capabilities

### 2. Intelligent Guidance
- Provides Trae usage optimization tips
- Helps save usage limits with efficient workflows
- Suggests best practices for your specific project

### 3. Smart File Handling
- Before any edit, reads project index first
- Opens relevant files efficiently
- Minimizes unnecessary file operations

## Usage Examples:

- "Ganpati, mujhe search functionality dikhao"
- "Project structure samjhao"
- "Trae ki konsi settings use karun?"
- "Meri usage limit kaise bachao?"

## Trae Optimization Settings:

1. **Batch Tool Calls**: Multiple searches ek saath
2. **Efficient Searching**: Specific directories target karo
3. **Smart File Reading**: Sirf necessary lines padho
4. **Cached Operations**: Repeat searches avoid karo
